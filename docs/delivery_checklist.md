# 可交付项目验收清单

日期：2026-06-06

## 当前交付范围

本项目当前交付为本地 Web + 命令行版本，支持：

- 输入两支球队进行自动预测。
- API-Football 自动取数。
- 真实赛前比赛单入口，首次打开不自动保存演示预测。
- 胜平负市场概率和模型概率。
- 大小球、让球模拟舱方向。
- 1000 元启动资金和均注模拟。
- SQLite 保存预测运行记录。
- 健康检查接口。
- 橘色信号品牌与石墨灰玻璃工作台界面，分区承载预测、数据处理、模型审计和历史报告。
- 数据处理板块展示近期有效比赛、特征计算结果及线性趋势图。
- 中文 Excel / PDF 报告导出。
- API-Football 结构化中文错误提示和自动重试。
- API 原始数据快照保存。
- 结构化优先级庄家报价快照和赔率时点保存。
- 赛果同步、时间切分校准审计与回测模块，支持 ROI、最大回撤、Brier Score、Log Loss。
- 今日随机比赛只抽取赛前比赛。
- 今日甲级联赛赛前列表按北京时间抓取并以中文显示，选中后再生成预测。
- 单场默认深度模式，补抓双方近 10 场技术统计和事件数据；快速模式保留为额度紧张时的降级选项。
- 每日批量建库入口，可手动运行或安装 macOS 每天 08:30 自动任务。
- 缺少可靠球队强度数据时，模拟舱降级为观望。
- 胜平负必须同公司完整 1X2；大小球和让球盘口必须同公司、同盘口线成对，水位异常会被排除。
- 研究方向必须通过基础研究 EV、优势门槛和纸上 EV 三重闸门；未经 `pfinal` 验收不产生正式模拟信号。
- 当前实际市场基准胜平负分布发生重大模型分歧时，三类市场 EV 全部暂停公开展示，原始试算仅在报告审计附录留存。
- 批量赛事池支持指定多个 fixture_id、批次持久化、历史恢复、筛选、搜索和单场钻取。
- 批次可手动设为每日官方复盘批次，同一日期同一范围只保留一个官方批次。
- 新增赛后复盘页，直接展示已结算比赛、待结算比赛、EV 候选真实结果和中文 Excel 复盘导出。
- 新增模拟舱账本结算能力：正式纸上观察入账后，可在赛果同步时自动结算盈亏；当前正式 EV 未开放时不写入虚假下注。
- 双击启动脚本 `start_worldcup_predictor.command`。
- 双击停止脚本 `stop_worldcup_predictor.command`。
- 首次安装脚本 `setup_worldcup_predictor.command`。
- 甲方交付说明 `交付说明.md`。
- 基础单元测试。

## 甲方需要提供

### 必需

- 真实要预测的赛事范围：世界杯正赛、预选赛、友谊赛，还是全部。

### 建议提供

- 偏好的庄家公司或赔率来源。
- 后续资金管理规则：固定均注、百分比均注、Kelly 折扣。
